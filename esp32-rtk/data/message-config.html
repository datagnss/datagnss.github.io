Message Configuration

The “Message Configuration” setting is used to configure the message configuration, including NMEA/RTCM message configuration.

  • NMEA: In general, we use NMEA messages output in Rover mode. We can enable the NMEA messages by checking the NMEA messages in the Message Configuration.
  • RTCM: In general, we use RTCM messages output in Base mode. We can enable the RTCM messages by checking the RTCM messages in the Message Configuration.
