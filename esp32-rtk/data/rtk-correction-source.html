RTK Correction Source

This panel is used to view/select the current RTK correction data source, to ensure the rover receives rtcm data.

  • Bluetooth: RTCM corrections data from Bluetooth connected mobile app. If you use app to connect the base staion, you can choose this option
  • External 6P Port: RTCM corrections data from External 6P Port connected to the telemetry radio.
  • NTRIP Client: RTCM corrections data from internal NTRIP Client connection. If you choose this option, please config the ntrip client information in below fields.
