RTK Correction Source

This panel is used to view/select the current RTK correction data source, to ensure the rover receives real-time differential information.

  • Bluetooth: RTCM corrections data from Bluetooth connected mobile app.
  • External 6P Port: RTCM corrections data from External 6P Port connected to the telemetry radio.
  • NTRIP Client: RTCM corrections data from internal NTRIP Client connection.
