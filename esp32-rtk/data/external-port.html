External Port Overview

External Port: JST-GH-1.25, 6P. This interface provides both power and serial signals, suitable for connecting to external devices such as telemetry radio.

  • Pin 1-2：Power/GND, please confirm voltage compatibility with the target device.
  • Pin 3-4：UART TX/RX, can be enabled in system settings for GNSS data output or pin swapping.
  • Pin 5-6：Reserved for compatibility and expansion (e.g., PPS, GPIO).
